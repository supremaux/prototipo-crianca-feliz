## 📱 Sobre o Projeto

O **Criança Feliz** é uma Single Page Application (SPA) desenvolvida para uma loja de moda infantil localizada em Campo Grande, MS. O projeto foi criado com foco em **mobile-first**, **velocidade** e **extrema facilidade de uso** para mães que navegam pelo celular.

### 🎯 Funcionalidades Principais

- ✨ **Vitrine de Produtos**: 8 cards de produtos fictícios com imagens, preços e categorias
- 🔍 **Página de Detalhes**: Visualização ampliada de cada produto com descrição baseada no framework AIDA
- 📱 **Design Mobile-First**: Layout totalmente responsivo e otimizado para dispositivos móveis
- 💬 **Integração WhatsApp**: Botão que envia mensagem personalizada com rich snippet e link do produto
- 🏷️ **Seletor de Tamanhos**: Opção para escolher o tamanho antes de enviar a mensagem
- 🎨 **Paleta Pastel**: Cores suaves e amigáveis que transmitem confiança e profissionalismo

### 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e moderna
- **JavaScript (Vanilla)** - Lógica da aplicação e manipulação do DOM
- **Tailwind CSS** - Estilização rápida e responsiva via CDN
- **Lucide Icons** - Ícones vetoriais modernos e leves
- **Google Fonts (Inter)** - Tipografia limpa e legível
- **Open Graph Tags** - Rich snippets para compartilhamento no WhatsApp
